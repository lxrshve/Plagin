# Unused Variable Detector (C/C++)

VS Code расширение для поиска неиспользуемых переменных в **C/C++** коде.
## Техническое задание
# Расширение VS Code "Детектор неиспользуемых переменных C++"
#Наименование
"Unused Variable Detector for C++"

##Назначение
Расширение для Visual Studio Code для автоматического обнаружения и визуального выделения неиспользуемых переменных в коде на C++.

## Основной функционал

- **Анализ в реальном времени** при редактировании C++ файлов
- **Обнаружение объявлений переменных** трех типов:
- **Отслеживание использований** переменных во всем файле

# Визуальное выделение
- **Подсветка фона** неиспользуемых переменных полупрозрачным красным цветом
- **Отображение рамки** красного цвета вокруг переменной
- **Добавление текстовой метки** "← UNUSED" после переменной
- **Автоматическое обновление** подсветки при изменениях в коде

- **Только для C/C++** (не для JavaScript/TypeScript)
- **Не работает автоматически** - требует ручного запуска
- **Не требует установки через Marketplace**

## Запуск в режиме разработки

1. Откройте проект расширения в VS Code
2. Нажмите **F5** или **Command+Shift+D** → "Run and Debug"
3. Откроется новое окно VS Code с активным расширением
4. В новом окне откройте C/C++ файл для анализа

## Использование

После открытия нового окна:
- Расширение активируется в новом окне автоматически при открытии `.c`/`.cpp`/`.h` файлов
